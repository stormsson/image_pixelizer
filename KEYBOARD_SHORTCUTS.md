# Keyboard Shortcuts

This document lists all keyboard shortcuts available in the Image Pixelizer application.

## File Operations

- **Ctrl+O** (Windows/Linux) or **Cmd+O** (macOS): Open/Load an image file
  - Opens a file dialog to select an image file (JPEG, PNG, GIF, BMP, WebP)
  - Displays the selected image in the main content area

- **Ctrl+S** (Windows/Linux) or **Cmd+S** (macOS): Save processed image
  - Opens a file save dialog to save the current processed image
  - Images are always saved in PNG format to preserve quality and transparency
  - Only available when an image is loaded

## Dialog Operations

- **Esc**: Close error dialogs and message boxes
  - Dismisses any open error or information dialogs

## Accessibility

All menu items and controls are accessible via keyboard navigation:
- Use **Tab** to navigate between controls
- Use **Enter** or **Space** to activate buttons
- Use **Arrow keys** to adjust slider values when focused
- Use **Alt+F** to open the File menu (Windows/Linux)
- Use **Ctrl+F** to open the File menu (macOS)

## Tips

- Keyboard shortcuts follow standard desktop application conventions
- All shortcuts are displayed in the menu bar next to their corresponding menu items
- Shortcuts are platform-aware (Ctrl on Windows/Linux, Cmd on macOS)
